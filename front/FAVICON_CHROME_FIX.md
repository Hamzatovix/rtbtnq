# Исправление маленького favicon в Chrome на десктопе

## Проблема
Иконка в открытой вкладке Chrome на десктопе слишком маленькая.

## Решение

### 1. Увеличены размеры favicon
- `favicon.ico`: увеличен до **128x128** (было 64x64)
- `favicon-128x128.png`: создан для Chrome
- `favicon-256x256.png`: создан для Chrome высокого разрешения
- `favicon.svg`: SVG версия (масштабируется автоматически) - **приоритетная**

### 2. Настройки в `layout.tsx`
SVG favicon стоит **первым** в списке иконок, что означает, что современные браузеры (включая Chrome) будут использовать его в первую очередь.

### 3. Очистка кеша Chrome

**Важно:** Chrome агрессивно кеширует favicon. Для применения изменений нужно:

#### Способ 1: Жесткая перезагрузка страницы
1. Откройте сайт в Chrome
2. Нажмите `Ctrl + Shift + R` (Windows/Linux) или `Cmd + Shift + R` (Mac)
3. Это очистит кеш и загрузит новую версию favicon

#### Способ 2: Очистка кеша через DevTools
1. Откройте DevTools (`F12`)
2. Правой кнопкой мыши на кнопку обновления страницы
3. Выберите "Очистить кеш и жесткая перезагрузка"

#### Способ 3: Полная очистка кеша Chrome
1. Откройте `chrome://settings/clearBrowserData`
2. Выберите "Изображения и файлы в кеше"
3. Выберите период "За все время"
4. Нажмите "Удалить данные"

#### Способ 4: Прямая ссылка на favicon
Откройте в новой вкладке:
```
https://rosebotanique.store/favicon.svg
```
Это заставит Chrome загрузить новую версию.

### 4. Проверка favicon

После очистки кеша проверьте:
1. Откройте сайт в Chrome
2. Проверьте вкладку браузера - иконка должна быть крупнее
3. Если все еще маленькая, проверьте:
   - Откройте `chrome://favicon/https://rosebotanique.store/`
   - Должна отображаться SVG версия

### 5. Технические детали

**Приоритет иконок в Chrome:**
1. `favicon.svg` (SVG - масштабируется автоматически) ✅ **ПРИОРИТЕТ**
2. `favicon-256x256.png` (256x256 для высокого разрешения)
3. `favicon-128x128.png` (128x128 стандартный размер)
4. `favicon.ico` (128x128 fallback для старых браузеров)

**Размеры файлов:**
- `favicon.svg`: ~31 KB (векторная графика)
- `favicon-256x256.png`: ~XX KB
- `favicon-128x128.png`: ~XX KB
- `favicon.ico`: ~XX KB

## После деплоя на сервер

После загрузки изменений на сервер:
1. Очистите кеш Chrome (см. выше)
2. Перезагрузите страницу с `Ctrl + Shift + R`
3. Проверьте, что иконка отображается правильно

## Если проблема сохраняется

1. Проверьте, что файлы загружены на сервер:
   - `https://rosebotanique.store/favicon.svg`
   - `https://rosebotanique.store/favicon-256x256.png`
   - `https://rosebotanique.store/favicon-128x128.png`

2. Проверьте заголовки ответа сервера:
   - Должен быть правильный `Content-Type` для SVG
   - Не должно быть агрессивного кеширования

3. Проверьте консоль браузера на ошибки загрузки favicon
